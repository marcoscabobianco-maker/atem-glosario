# Libro Azul (Custodializado)

## Presentación

- **Status tras la incursión al plano astral:** Parcialmente investigado via [[Akashic_Records|Akashic Records]]
- **Contenido descubierto:** Frontiticios mágicos (Speculum Artis, integración Naturaleza-Magia), escritura doble (dos autores: [[Pablo_de_Alejandria|Pablo]] + hijo?), contiene "secretos oscuros de [[Cronomon]]"
- **Hallazgo paradoja:** [[Cronomon]] escribió libro pero "partes censorizadas" — Libro tiene censura que grupo debe desentrañar
- **Ediciones:** "Doble y extrañamente escrita," múltiples ediciones (posiblemente cada viaje temporal crea versión nueva)
- **Ubicación tras la incursión astral:** El grupo retiene custodia; los Caballeros tienen acceso para consulta
- **Investigación restante:** Sector inferior del Libro (presumiblemente al llegar a París Ucrónica), acceso a "Novena Puerta" (última verdad cosmológica)
