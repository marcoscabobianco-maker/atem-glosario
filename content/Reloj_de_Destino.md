# Reloj de Destino

## Presentación

**[[Propietario]]:** [[Michel|Michel]] (presumiblemente aún en posesión cuando capturado)
**Poderes:**
 - **detención del tiempo:** Detiene tiempo de manera local. Uso 1x/month
 - **True Resurrection:** Resurrección verdadera. Pendiente activación. Uso 1x/month
 - **Wielder:** Cambio portador. Uso 1x/month

**Mecánica limitación:** Portador nuevo requiere precaución. "Use Magic Device" puede sortear restricciones (como hizo [[Michel|Michel]] antes).

**Riesgo:** Doble portador activa contingencias desconocidas. "Hay un True Resurrection pendiente sobre vosotros" sugiere deuda cósmica.

**Implicación en el último encuentro templario:** El reloj es presumiblemente clave para vencer a [[Cronomon]] (la detención del tiempo puede detener maniobras temporales del antagonista) y para la potencial resurrección de [[Arsvan|Ärsvan]] (en aquel encuentro Ärsvan se sacrifica).
